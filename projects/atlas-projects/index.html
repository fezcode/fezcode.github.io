About the Project

01

Overview

The Atlas Suite is a family of minimalist, high-performance command-line tools built with Go. Each tool is designed to solve one problem exceptionally well, adhering to the Unix philosophy and leveraging the power of Bubble Tea for beautiful TUIs.

02

Tools in the Suite

    1. atlas.todo: A fast, minimalist, local-first TUI task manager. Features Vim-bindings, smart grouping, and CLI quick-add support.
    1. atlas.stats: Real-time system monitoring. View CPU, memory, disk, and network usage in a responsive terminal interface.
    1. atlas.websearch: A blazing fast, interactive CLI search tool. Search DuckDuckGo, Wikipedia, Hacker News, and Reddit with zero configuration.
    1. atlas.compass: A secure, local-first terminal password manager. Features AES-256-GCM encryption and Argon2id key derivation for high-stakes security.
    1. atlas.clock: A high-visibility world clock dashboard. Features millisecond-precision real-time counters and a searchable IANA timezone list.
    1. atlas.cam: A terminal webcam viewer and ASCII camera. Features real-time edge detection, image filters, and GIF recording.
    1. atlas.games: A terminal-based game collection. Featuring Wilson's Revenge directly in your shell.
    1. atlas.bench: A high-performance benchmarking tool for terminal applications and command-line processes. Measure CPU, memory, and latency with millisecond precision.
    1. atlas.radar: A high-performance Git workspace monitor. Scan, monitor, and sync multiple repositories simultaneously with a beautiful, minimalist CLI.

03

Why Atlas?

  • Go-Powered: Built for speed and portability.
  • Beautiful TUIs: Uses Bubble Tea and Lip Gloss for a modern terminal experience.
  • Minimalist: No bloated interfaces, just the features you need.
  • Local-First: Your data stays on your machine.

Installation

Each tool can be installed using gobake for a native build experience:

# Example for atlas.todo
git clone https://github.com/fezcode/atlas.todo
cd atlas.todo
gobake build
The same process applies to atlas.stats, atlas.websearch, atlas.compass, atlas.clock, atlas.cam, atlas.games, atlas.bench, and atlas.radar. To build for all platforms, simply run gobake build in any of the tool repositories.
