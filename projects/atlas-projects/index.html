About the Project

01

Overview

The Atlas Suite is a family of minimalist, high-performance command-line tools built with Go. Each tool is designed to solve one problem exceptionally well, adhering to the Unix philosophy and leveraging the power of Bubble Tea for beautiful TUIs.

02

Tools in the Suite

    1. atlas.hub: The centralized installer and manager for the Atlas Suite. Discover, build, and maintain your entire Atlas toolkit from a single, unified interface.
    1. atlas.todo: Fast, keyboard-centric terminal user interface for task management. Features Vim-bindings, smart grouping, and CLI quick-add support.
    1. atlas.stats: Fast, interactive terminal user interface for real-time system monitoring. View CPU, memory, disk, and network usage.
    1. atlas.websearch: Fast, interactive terminal user interface for web searching across DuckDuckGo, Wikipedia, Hacker News, and Reddit.
    1. atlas.compass: Secure, local-first password manager for the terminal featuring AES-256-GCM encryption.
    1. atlas.clock: High-visibility, multi-timezone dashboard with millisecond precision and searchable IANA timezone list.
    1. atlas.cam: Terminal webcam viewer and ASCII camera with real-time edge detection and filters.
    1. atlas.games: Terminal-based game collection including Wilson's Revenge and Wave Function Collapse.
    1. atlas.bench: High-precision, multi-command benchmarking tool with visual TUI comparison and millisecond accuracy.
    1. atlas.radar: A tool to monitor git statuses across multiple repositories simultaneously.
    1. atlas.otp: High-visibility, minimalist terminal TOTP (2FA) manager with a sleek Onyx & Gold theme.
    1. atlas.diff: High-visibility, side-by-side terminal diff tool for file comparison.
    1. atlas.screensaver: A collection of nostalgic and aesthetic terminal screensavers (Pipes, Stars, Matrix, DNA, Waves).
    1. atlas.facade: Retro-future Pip-Boy style mock API server for instant frontend prototyping.
    1. atlas.pq: Minimalist PIML processor (inspired by jq) for slicing, filtering, and mapping data.
    1. atlas.hash: Fast, minimalist terminal utility for generating and comparing file hashes (MD5, SHA1, SHA256, SHA512).
    1. atlas.grave: High-fidelity interactive process reaper with a Pip-Boy inspired TUI.
    1. atlas.radio: Global terminal radio receiver with access to thousands of live stations.
    1. atlas.deck: Interactive TUI command deck for project workflows and automation through a grid of "pads."
    1. atlas.convert: Image conversion tool for JPEG, PNG, and HEIC formats.
    1. atlas.conquistador: A powerful and beautiful terminal-based file explorer like Finder or Windows Explorer.
    1. atlas.horizon: High-fidelity environmental and weather dashboard with atmospheric monitoring.
    1. atlas.cat: A beautiful, high-performance terminal text viewer with syntax highlighting and line numbers.
    1. atlas.ed: A beautiful, high-performance terminal text editor with syntax highlighting, line numbers, and search.
    1. atlas.gitty: Comprehensive TUI git client with graphs, branch, tag, and multi-repo management.
    1. atlas.ip: Fast, minimal tool to fetch your local and public IP address with geolocation data.
    1. atlas.sand: Interactive terminal-based falling sand and particle physics simulator.
    1. atlas.sql: Terminal-based SQL client for SQLite and PostgreSQL with syntax highlighting.
    1. atlas.color: Interactive TUI color picker and converter (Hex, RGB, HSL).
    1. atlas.archive: An interactive step-by-step archiver and extractor for managing compressed files.
    1. atlas.burner: A beautiful TUI image burner application for OS images and USB drives.
    1. atlas.batchdown: A powerful batch downloader that processes multiple URLs from text files sequentially.
    1. atlas.guide: A personal health and wellness tracker for calories, gym activities, and mood.
    1. atlas.notes: A beautiful terminal-based markdown note manager with built-in editor and high-fidelity rendering.

03

Why Atlas?

  • Go-Powered: Built for speed and portability.
  • Beautiful TUIs: Uses Bubble Tea and Lip Gloss for a modern terminal experience.
  • Minimalist: No bloated interfaces, just the features you need.
  • Local-First: Your data stays on your machine.
