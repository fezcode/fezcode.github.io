LABEL: Examples
SUBTEXT: Comparative Exhibition

Compare the elegance of PIML against traditional formats. Notice how the vertical flow of PIML reduces cognitive load compared to the nested brackets of JSON.

### PIML Schema
```piml
(project)
  (name) piml
  (version) 1.0.0
  (description) Parenthesis Intended Markup Language
  (author)
    (name) fezcode
    (github) https://github.com/fezcode
  (tags)
    > markup
    > data-serialization
    > minimalist
  (isActive) true
  (spec)
    (version) 4.3
    (status) stable
```

### JSON Legacy
```json
{
  "project": {
    "name": "piml",
    "version": "1.0.0",
    "description": "Parenthesis Intended Markup Language",
    "author": {
      "name": "fezcode",
      "github": "https://github.com/fezcode"
    },
    "tags": [
      "markup",
      "data-serialization",
      "minimalist"
    ],
    "isActive": true,
    "spec": {
      "version": 4.3,
      "status": "stable"
    }
  }
}
```
